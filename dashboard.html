Lisää Kulova nettisivullesi

Kopioi tämä koodi nettisivusi <body>-tagin loppuun. Chat-nappi ilmestyy oikeaan alakulmaan.

—

Asennusohjeet

1

Kirjaudu WordPress-hallintapaneeliin ja mene Ulkoasu → Muokkaa teemaa tai asenna plugin Header Footer Code Manager.

2

Etsi footer.php-tiedosto ja lisää koodi juuri ennen </body>-tagia.

3

Kopioi alla oleva koodi ja liitä se sinne. Tallenna.

1

Avaa Wix Editor ja mene Asetukset → Mukautettu koodi (Settings → Custom Code).

2

Klikkaa + Lisää koodi ja valitse sijainti: Body - end.

3

Liitä koodi kenttään ja valitse Kaikki sivut. Tallenna ja julkaise.

1

2
